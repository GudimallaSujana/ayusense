🌿 Ayusense – AI Ayurvedic Wellness Platform

Ayusense is an intelligent web application that combines ancient Ayurvedic knowledge with modern AI to provide personalized health insights, plant identification, and natural remedies.

🚀 Features 🌱 1. Identify Medicinal Plants Upload a plant image AI analyzes and identifies the plant Returns: Scientific name Key features Health benefits Ayurvedic uses Home remedies

🩺 2. Find Ayurvedic Remedies Enter symptoms (e.g., fever, cough, cold) Optional location input AI provides: Recommended medicinal plants Remedies with preparation steps Precautions Lifestyle & diet suggestions

🤖 AI-Powered Backend Primary: Google Gemini API Fallback: OpenAI API Ensures reliability even if one provider fails 🛠️ Tech Stack Frontend React + TypeScript TanStack Router Tailwind CSS Backend Server functions (TanStack Start) AI Integration: Google AI Studio OpenAI Optional (Future) Supabase (auth, storage, DB)
